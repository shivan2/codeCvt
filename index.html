카포 코드 변환기

곡에 포함된 코드를 여러 개 선택해두고, 카포 프렛만 바꾸면 실제로 잡아야 할 코드가 바로 갱신됩니다.

코드 선택

루트음/코드타입을 따로 조합하지 않고, 자주 쓰는 코드를 여러 개 동시에 켜고 끌 수 있습니다.
- 코드를 짧게 클릭시 -> 코드 선택
- 코드를 길게 클릭시 -> 코드 타입 선택(m, 7, sus4 등..)

선택/해제만 하면 아래 결과가 자동 반영됩니다.

카포 프렛

버튼으로 프렛을 조절하면 결과가 즉시 다시 계산됩니다.

현재 카포
0F

선택한 원곡 코드

카포 적용 코드

현재 버전은 샵 표기 기준입니다. 다음 단계에서 Bb, Eb 같은 플랫 표기와 고급 코드 조합 모드를 별도로 확장 예정입니다.

코드 타입 선택

C
코드 타입을 선택하면 즉시 추가됩니다.

베이스 음 선택

C/E
베이스 음을 고르면 미리보기가 갱신됩니다.
